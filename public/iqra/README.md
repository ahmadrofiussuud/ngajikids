# Iqra PDF Files Directory

Silakan simpan 6 file PDF Iqra Anda di folder ini dengan nama-nama berikut agar dapat dibaca oleh sistem:

- `jilid-1.pdf` (untuk Iqra Jilid 1)
- `jilid-2.pdf` (untuk Iqra Jilid 2)
- `jilid-3.pdf` (untuk Iqra Jilid 3)
- `jilid-4.pdf` (untuk Iqra Jilid 4)
- `jilid-5.pdf` (untuk Iqra Jilid 5)
- `jilid-6.pdf` (untuk Iqra Jilid 6)

### Contoh Full Path:
`public/iqra/jilid-1.pdf`
`public/iqra/jilid-2.pdf`
...
`public/iqra/jilid-6.pdf`
